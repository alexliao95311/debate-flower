# Flower

A fork of [Ashwagandhae's Debate Flower](https://debate-flow.vercel.app) — added account sign-in and fixed share features.

Created by Alex Liao.

[Original](https://debate-flow.vercel.app) · [This fork](https://github.com/alexliao95311/debate-flower)

## Usage

```bash
git clone https://github.com/alexliao95311/debate-flower.git
cd debate-flower
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
